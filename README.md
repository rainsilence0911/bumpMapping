# bumpMapping
基于webgl和vue2实现bump mapping算法(内附效果图)

bump mapping是一种廉价的实现凹凸纹理的算法。这个算法不需要为了凹凸效果去改变顶点坐标，所有的纹理完全在着色器内部生成
